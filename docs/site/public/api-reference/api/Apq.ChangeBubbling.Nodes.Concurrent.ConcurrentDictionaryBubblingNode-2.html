基于 ConcurrentDictionary<TKey, TValue> 的线程安全更改事件冒泡节点。 直接使用 ConcurrentDictionary 的原子操作，手动触发变更事件。 适用于任意 TValue 类型（无需实现 IChangeNode）。 所有操作都是线程安全的。

public class ConcurrentDictionaryBubblingNode<TKey, TValue> : ChangeNodeBase, IChangeNode, IBubblingChangeNotifier, INotifyPropertyChanged where TKey : notnull
