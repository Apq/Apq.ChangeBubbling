BeginBatch()

开始批量操作。在批量操作期间，变更事件会被收集而不是立即触发。 支持嵌套调用，只有最外层的 EndBatch 才会触发事件。

public void BeginBatch()

BeginCoalesce()

开始事件合并模式。在此模式下，相同属性名的变更事件会被合并， 只保留最新的值（OldValue 保留第一次的，NewValue 使用最后一次的）。 适用于高频属性更新场景，减少下游处理压力。

public void BeginCoalesce()
