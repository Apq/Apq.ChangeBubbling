Namespace Apq.ChangeBubbling.Nodes

Classes

DictionaryBubblingNode<TKey, TValue>

基于 IDictionary 的更改事件冒泡节点。 使用 ObservableCollectionAdapter 代理底层集合， 将 Add/Remove/Indexer 等操作映射为 BubblingChange 并向上冒泡。 适用于任意 TValue 类型（无需实现 IChangeNode）。

注意：此类非线程安全。如需线程安全版本，请使用 ConcurrentDictionaryBubblingNode<TKey, TValue>。

ListBubblingNode<T>

基于 IList 的更改事件冒泡节点。 使用 ObservableCollectionAdapter 代理底层集合， 将 Add/Insert/Remove/Indexer 等操作映射为 BubblingChange 并向上冒泡。 适用于任意 T 类型（无需实现 IChangeNode）。

注意：此类非线程安全。如需线程安全版本，请使用 ConcurrentBagBubblingNode<T>。
